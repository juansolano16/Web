** Inicializar Crear Proyecto
django-admin startproject WebProduct

	** Crear Aplicacion
	python manage.py startapp WebProduct

	** Verificar si hay errores en la App
	python manage.py check WebProduct

	** Crear tablas de la App
	python manage.py makemigrations

	** Generar Datos BD
	python manage.py sqlmigrate WebProduct 0001

	** Editar Tablas 
	python manage.py shell
	>>> from WebProduct.models import Articulo
	>>> art = Articulo(nombre='mesa', seccion='decoracion', precio=12)
	>>> art.save()

	Equivalente:
	>>> art2 = Articulo.objects.create(nombre='taladro', seccion='ferreteria', precio=13)

	Sentencia Sql:
	>>> sql =  Articulo.objects.all()
	>>> sql.query.__str__()
	'SELECT "WebProduct_articulo"."id", "WebProduct_articulo"."nombre", "WebProduct_articulo"."seccion", "WebProduct_articulo"."precio" FROM "WebProduct'

	** Efectuar models sobre tabla existentes
	python manage.py migrate --fake <appname>
	python manage.py inspectdb > model.py ##Para revisar los models de una base de datos
